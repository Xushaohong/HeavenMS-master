# 个人私服冒险岛(based upon HeavenMS)


一款85~00后(但不限于任何年龄段)心中的当年红极一时的情怀向网(gan)游的私服化, 面向所有喜爱冒险岛的朋友开放之作,绝对无任何商业利益.作者本人会不定期调试,更新游戏机制,内容.将最好的冒险岛旧酒换新瓶的体验带给大家!
